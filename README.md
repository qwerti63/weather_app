# weather_app

# Погода Про (Weather pro)
Стильное, легковесное приложение для проверки текущей погоды и прогноза на 3 дня. Написано на Python с использованием современного графического интерфейса **CustomTkinter** и бесплатного API **Open-Meteo**.


## Особенности
* **Умный поиск:** Ввод городов на любом языке (включая русский).
* **Прогноз:** Отображение максимальной температуры на 3 дня вперед.
* **Дизайн:** Современный графический интерфейс с поддержкой темной темы.
* **Портативность:** Компилируется в один единственный исполняемый файл, работающий без установки Python.

## Быстрый запуск
1. **Клонируйте репозиторий:**
    git clone https://github.com/qwerti63/weather_app
2. **Установите зависимости:**
    pip install -r requirements.txt
3. **Запустите приложение:**
    python weather.py

## Сборка в один файл (Linux бинарник)
Чтобы собрать самодостаточное приложение, запустите PyInstaller:
    pyinstaller --onefile --windowed --add-data "icon.png:." --copy-metadata customtkinter weather.py
**Готовый файл появится в папке dist/.**

## Инструкция для пользователей (Как запустить скачанный файл)
Так как Linux защищает систему от файлов из интернета, после скачивания файла weather сделайте следующее:
 1. Откройте терминал в папке с файлом и пропишите:
     chmod +x weather
 2. Либо нажмите правой кнопкой мыши > **Свойства** -> **Права** -> поставьте галочку **«Разрешить выполнение файла как программы»**.
 3. Запустите двойным кликом!
